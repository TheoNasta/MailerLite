# Live Demo:
https://theonasta.github.io/MailerLite/dist/

# With Docker:

`bash build.sh` - Setup the image
`bash develop.sh` - Start live reloading

You can adjust the port in `docker-compose.yaml` if the current one is taken on your machine.

# Without Docker:

- Install node
- Install yarn
- `yarn`
- `yarn run start`
